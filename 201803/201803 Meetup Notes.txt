## Presentations
Both Fareeza and Mostafa walked through a common example using the same StatCan census data.  This will allow us to see & compare the features of each language & environment.


### R
- Using R Studio to develop and present the results in a notebook format
- The data being used has been made available through a great site: https://censusmapper.ca
	- to use this site yourself, register for an API KEY
- there are tricks to installing and having packages run in the windows "user space"
- Visualizing and mapping data to understand your data

- Code and output can be found in Fareeza's Github Repo:
	- https://github.com/statgeek/R-Tutorials

### Python
- Using Jupyter to develop and present in a notebook format
- Showing the features of data cleanup and visualizing
- Custom packages can be written and integrated 
	- use PYTHONPATH environment variable to point to directory of your code
 
- Code and output can be found in Mostafa's Github Repo:
	- https://github.com/statgeek/R-Tutorials
